Setting up Videos for Sleepover

Basically, in order to load any of the above in game, download any of the files you wish to be displayed for you in game, then, do as followed for each :

Open the path to your roblox clients folder, typically located under %LocalAppData%\Roblox\Versions, the client itself should be in the folder thats offset after version-2, if not, one of them should be it, just make sure its not roblox studio.

After, go into the folder called content, If you have a folder named TvVideos

then, just drag the video/s in there and youre ready to join the game!

If you don't,

then, create the folder with the same name(Case sensitivity matters) and drag the video/s inside of it, and nooow youre done!
